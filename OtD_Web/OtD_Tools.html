Preparing your laptop

Below are instructions to install the tools used in this course and for windows users a set of command line tools.

Tools to install:

  • R and RStudio Desktop (select RStudio in the Products menu, then download RStudio Desktop)
  • jq: a lightweight JSON query tool. Cygwin users and Koding.com users should
    1. download the following script to their environments with 'wget http://humanities.uva.nl/~mkoolen1/teaching/data/FOtD/2015/scripts/get_jq.sh',
    2. make the script executable with 'chmod u+x get_jq.sh'
    3. run the script as administrator with 'sudo ./get_jq.sh'
  • git: a version control system for safely storing and sharing your work. (git installing instructions)

Windows users

Windows users need to install Cygwin (and include the Git package)

  • Go to the Cygwin website
  • Download the setup file (the 32-bit or 64-bit version, depending on the processor in your computer, which is either a 32-bit or 64 bit processor)
  • Double click the setup file and choose the default options.
  • For the mirror site, look for ftp://ftp.snt.utwente.nl
  • In the Select Packages screen, include the following packages:
